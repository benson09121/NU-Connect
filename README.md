## 📖 About the Project

**NU Connect** is a centralized digital platform designed to streamline the operations of Recognized Student Organizations (RSOs) at National University Dasmariñas. The system addresses common challenges in student organization management, including event coordination, financial tracking, attendance monitoring, and member engagement.

Traditional manual processes often lead to inefficiencies, lack of transparency, and miscommunication among student leaders and administrators. NU Connect solves these challenges by providing a structured, user-friendly platform that enhances organizational efficiency, accountability, and user engagement.

This project contributes to improving campus life and student development by leveraging modern web technologies to create a practical and scalable solution for higher education institutions.

### 🎯 Project Goals

- **Streamline Organization Operations** - Centralize RSO management processes
- **Enhance Event Coordination** - Simplify event planning and attendance tracking
- **Improve Financial Transparency** - Enable accurate financial recording and reporting
- **Boost Member Engagement** - Foster better communication and participation
- **Ensure Accountability** - Provide transparent workflows and approval processes

## ✨ Key Features

### Organization Management
- **📋 Organization Application System** - Digital application process for new RSOs
- **👥 Member Management** - Track organization members and roles
- **📊 Dashboard Analytics** - Comprehensive overview of organization activities
- **🔐 Secure Authentication** - Microsoft account integration for user verification

### Event Management
- **📅 Event Creation & Scheduling** - Plan and manage organization events
- **✅ Approval Workflows** - Structured event approval process
- **📍 Real-time Attendance Tracking** - QR and NFC technology integration
- **📱 Mobile-Friendly Interface** - Access events on any device

### Financial Management
- **💰 Financial Recording** - Track income and expenses
- **📈 Budget Monitoring** - Manage organization budgets
- **📑 Financial Reports** - Generate comprehensive financial statements
- **🔍 Transparency** - Clear audit trails for all transactions

### Administrative Features
- **🎓 RSO Oversight** - Monitor all student organizations
- **📝 Application Review** - Evaluate new organization applications
- **🔔 Notification System** - Real-time updates and alerts
- **📊 Analytics & Reporting** - Data-driven insights for decision-making

## 🛠️ Technology Stack

### Frontend
- **React.js** - Modern component-based UI framework
- **JavaScript** - Client-side functionality
- **Nginx** - Web server and reverse proxy

### Backend
- **Node.js** - Server-side runtime environment
- **Express.js** - Web application framework
- **SSE** - Real-time communication
- **Node-cron** - Scheduled task management

### Database & Caching
- **MySQL 8.4.6** - Relational database management
- **Redis** - In-memory caching and session storage

### Authentication & Security
- **Microsoft Azure MSAL** - Microsoft account authentication
- **JWT (JSON Web Tokens)** - Secure token-based authentication
- **Helmet.js** - Security headers middleware
- **Express Rate Limit** - API rate limiting

### DevOps & Deployment
- **Docker & Docker Compose** - Containerization and orchestration
- **Azure** - Cloud hosting platform
- **Cloudflare** - CDN and security services

### Additional Technologies
- **QR & NFC** - Attendance tracking technologies
- **Multer** - File upload handling
- **Nodemailer** - Email notifications
- **Sharp** - Image processing

## 🌐 Deployment

The system is designed for deployment on **Azure** with:
- **Docker containerization** for consistent environments
- **Cloudflare** for CDN and DDoS protection
- **Nginx** as reverse proxy and static file server
- **Redis** for session management and caching
